====================================================================================================
VISÃO GERAL
====================================================================================================
    Este documento explica como configurar o ambiente para execução do ANTLR no seu Windows.



====================================================================================================
REQUERIMENTOS
====================================================================================================
    - Complete ANTLR 4.5.3 Tool
    - Java Development Kit (JDK) Versão 6 ou Superior
    - Eclipse IDE



====================================================================================================
CONFIGURAÇÃO DO AMBIENTE
====================================================================================================
    1. É necessário especificar temporariamente na variável de ambiente CLASSPATH o caminho do 
    ANTLR. Para isto, execute o comando abaixo substituindo o caminho 
    C:\compilers-cin\aulas-praticas\ap3\antlr-4.5.3-complete.jar pelo caminho do ANTLR do seu
    computador.
    
    > SET CLASSPATH=.;C:\compilers-cin\aulas-praticas\ap3\antlr-4.5.3-complete.jar;%CLASSPATH%



====================================================================================================
GERAÇÃO DOS LEXERS, PARSERS E VISITORS DA GRAMÁTICA
====================================================================================================
    Como exemplo para essa etapa, utilizaremos a gramática Cymbol que se encontra no diretório 
    compilers-cin\aulas-praticas\ap3\antlr-cymbol-grammar-checker-types\.
    
    1. Abra a linha de comando do Windows (cmd.exe) no diretório compilers-cin\aulas-praticas\ap3\antlr-cymbol-grammar-checker-types\.
    2. Execute o comando abaixo para gerar os lexers, parsers e visitors em Java no pacote autogenerated:
    
    > java org.antlr.v4.Tool Cymbol.g4 -visitor -no-listener -package autogenerated
    
    3. Uma interface visitor será gerada (CymbolVisitor), bem como uma implementação padrão dela 
    (CymbolBaseVisitor) além dos tokens e de outros arquivos em Java.
